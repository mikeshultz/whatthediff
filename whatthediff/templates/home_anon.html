What is WhatTheDiff?

WhatTheDiff is a system for tracking changes to content on Web documents. What we mean by content is the actual text and data on the page itself. The changes to style an appearence generally aren't bothered with here unless it alters the structure of the document.

What is WTD good for?

  • Seeing the differences between different versions of a Web page
  • Keeping track of data displayed on a page or simple REST API call
  • Watching for when content is updated on wikis
  • Being notified when a friend updates their blog
